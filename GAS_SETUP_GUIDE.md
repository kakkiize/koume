# 🌸 梅乃園幼稚園 成長チェックアプリ — GAS セットアップ手順書

## 概要

このアプリは2つのモードで動作します：

| モード | 説明 | データ保存先 |
|--------|------|-------------|
| **GitHub Pages モード** | ブラウザで直接アクセス | ブラウザの localStorage |
| **GAS Webアプリ モード** | Google Apps Script 経由でアクセス | Googleスプレッドシート |

スプレッドシートとの連動には **GAS Webアプリ モード** が必要です。以下の手順でセットアップしてください。

---

## ステップ 1：スプレッドシートを開く

以下のURLにアクセスしてください：

**スプレッドシートURL：**
```
https://docs.google.com/spreadsheets/d/1DWmMVyHSyxkC3WMgIM3CHRsMlT1m75CL1zXEUvhP02o/edit
```

---

## ステップ 2：Apps Script エディタを開く

1. スプレッドシートを開く
2. メニューから **「拡張機能」→「Apps Script」** をクリック
3. Apps Script エディタが新しいタブで開きます

---

## ステップ 3：Code.gs を貼り付ける

1. エディタ左側の「コード.gs」をクリック
2. 既存の内容をすべて削除
3. 以下のファイルの内容をコピーして貼り付ける：

   **ファイル：** `Code.gs`（GitHubリポジトリに含まれています）

4. **Ctrl+S**（Mac: Cmd+S）で保存

---

## ステップ 4：index.html を追加する

1. エディタ左側の「+」ボタンをクリック
2. 「HTML」を選択
3. ファイル名を `index` と入力（`.html` は自動付与）
4. 既存の内容をすべて削除
5. `index.html` の内容をコピーして貼り付ける
6. **Ctrl+S** で保存

---

## ステップ 5：初期セットアップを実行する

1. エディタ上部の関数選択ドロップダウンで **`setupSheets`** を選択
2. **「実行」**ボタン（▶）をクリック
3. 初回は権限の確認が表示されます：
   - 「権限を確認」をクリック
   - Googleアカウントを選択
   - 「詳細」→「梅乃園成長チェックアプリ（安全ではないページ）に移動」をクリック
   - 「許可」をクリック
4. 実行完了後、スプレッドシートに以下のシートが自動作成されます：
   - `保護者リスト`
   - `チェック記録`
   - `感想記録`

---

## ステップ 6：Webアプリとしてデプロイする

1. エディタ右上の **「デプロイ」→「新しいデプロイ」** をクリック
2. 「種類の選択」で **「ウェブアプリ」** を選択
3. 以下のように設定：

   | 設定項目 | 設定値 |
   |---------|--------|
   | 説明 | 梅乃園成長チェックアプリ v1.0 |
   | 次のユーザーとして実行 | **自分（メールアドレス）** |
   | アクセスできるユーザー | **梅乃園幼稚園ドメインのユーザー全員** または **全員（Googleアカウントを持つユーザー）** |

4. **「デプロイ」** をクリック
5. 表示される **「ウェブアプリのURL」** をコピーして保存

   例：`https://script.google.com/macros/s/XXXXXXXXXX/exec`

---

## ステップ 7：保護者にURLを共有する

デプロイ後に取得したURLを保護者に共有してください。

> ⚠️ **重要：** GAS Webアプリとして動作させるには、保護者が**Googleアカウントでログイン**した状態でアクセスする必要があります。

---

## データの確認方法

アプリを使用すると、スプレッドシートに以下のデータが自動保存されます：

### 保護者リスト シート
| 列 | 内容 |
|----|------|
| A | メールアドレス |
| B | お子様の名前 |
| C | よみがな |
| D | クラス（赤小梅 / 白小梅） |
| E | 登録日 |

### チェック記録 シート
| 列 | 内容 |
|----|------|
| A | メールアドレス |
| B | カテゴリID |
| C | 項目番号 |
| D | チェック日インデックス（0-4） |
| E | スコア（1-4） |
| F | 更新日時 |

### 感想記録 シート
| 列 | 内容 |
|----|------|
| A | メールアドレス |
| B | チェック日インデックス（0-4） |
| C | 感想テキスト |
| D | 更新日時 |

---

## 集計シートの生成

チェック日が完了すると自動的に集計シートが更新されます。手動で更新する場合：

1. Apps Script エディタで **`generateSummarySheets`** を実行
2. スプレッドシートに以下のシートが生成されます：
   - `赤小梅特別シート`
   - `白小梅特別シート`

---

## 毎朝の自動更新設定（オプション）

毎朝6時に自動で集計シートを更新する場合：

1. Apps Script エディタで **`setupDailyTrigger`** を実行
2. 「毎朝6時の自動更新トリガーを設定しました！」と表示されれば完了

---

## トラブルシューティング

| 問題 | 解決方法 |
|------|---------|
| 「シートが見つかりません」エラー | `setupSheets()` を実行してシートを初期化する |
| 保護者がアクセスできない | デプロイ設定の「アクセスできるユーザー」を確認する |
| データが保存されない | GASの実行権限が正しく設定されているか確認する |
